# PWC Routes
PWC Routes Backend Coding Challenge


## Running the app
- Clone the repository:
```
git clone https://github.com/bogab/routes.git
```

- Go into `route` directory:
```
cd route
```

- Build the project:
```
./mvnw clean install
```

- Run the application:
```
./mvnw spring-boot:run
```

- To run the tests:
```
./mvnw test
```

- Perform requests:

```
curl http://localhost:8080/routing/CZE/ITA
```
